# Lost in Futility

Lost in Futility is a one-month collaborative game project created by my friend and me. Developed under a tight deadline, it showcases our teamwork, rapid iteration, and focus on building an atmospheric experience from a shared creative vision.

## Play the Game
Lost in Futility is available to play on itch.io:
